## 一、看视频学到什么

视频 UP 主：零界世访

### 核心方法论：岗位证据法

三个问题：
1. 我要进入哪一类岗位？
2. 这个岗位要我交付什么结果？
3. 我现在有什么作品或经历能证明我做得到？

四个步骤：
1. 去招聘平台找 3 个真实想投的岗位
2. 每个岗位拆三列：必须会的技术 / 必须交付的结果 / 我现在能拿出什么
3. 找到最大能力缺口，定下接下来一个月做什么
4. 用学生优惠云/GitHub Pages 把作品发布出去

### 关键反例（视频里的 A 同学）

> "我学了 Java、Spring Boot、MySQL、Redis，刷了八股"

vs B 同学：

> "我做了宿舍维修报修系统——能登录、提交工单、分配维修人员、查看处理状态、写了 API 文档、部署到服务器、用 AI 生成测试用例并自己验收"

**A 和 B 的差距不是技术栈，是"可验证的作品"**。
