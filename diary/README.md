## 写日记的原则

1. **诚实优先**——承认卡住、承认拖延、承认放弃某个方案，不要写"一切顺利"
2. **决策可追溯**——为什么选 A 不选 B，写清楚
3. **每周日复盘**——用 `../templates/weekly-review-template.md` 模板
4. **不要写废话**——不要"今天学了 XX"，要"今天 commit 了 XX，让证据表从 Y 变成 Z"
