## 真实投递策略（v2.1 重要补充）

> **6-24 用户原话**："这些企业都是我随手找的，不一定是去这些，我只是看到它JD毕竟丰富，可以作为参考"

**这意味着**：
- **A2 包装是确定目标**（让 AI-food / HexoAIAgent 覆盖 3 类岗位的能力要求）
- **真实投递时**：从 BOSS / 牛客重新选公司，按"能力对照表"快速判断
  - "这家公司要 Redis + Docker + Swagger" → AI-food 已覆盖 → 立刻投
  - "这家公司要 Spring Cloud 微服务" → 暂缓，标记为 B1 完成后投

**新习惯**：每看到 1 个新 JD，**用对照表快速过一遍**——15 秒判断"能不能投"。
