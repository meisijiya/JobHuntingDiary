## 五、HexoAIAgent-Java 架构（langchain4j 主导）

```
用户提问（HTTP / SSE）
   ↓
Spring Boot Controller
   ↓
langchain4j Agent（编排）
   ↓
┌──────────┬──────────┬──────────┐
│ RAG 检索  │ 工具调用  │ 记忆系统  │
│ (博客 MD) │ (联网/计算)│ (会话级) │
└──────────┴──────────┴──────────┘
   ↓
LLM 生成答案（带引用 + 文章链接）
   ↓
结构化响应 + 日志记录
```

### Agent 工程 6 步 checklist（视频原话）

1. **定义边界**：能做什么（基于博客知识库的回答）/ 不能做什么（编造、未读文章、隐私问题）
2. **整理上下文**：博客目录解析 + Markdown 切片 + 来源标注
3. **设计工具调用**：搜索工具 / 重新检索工具 / 反馈工具
4. **人工确认**：答案来源不确定时请求人工确认
5. **评测**：典型 3 + 异常 3 + 边界 3 = **9 条样例**
6. **观测与复盘**：记录 query / 检索结果 / LLM 输出 / 用户反馈

### 技术选型

- **框架**：Spring Boot 3.x + langchain4j 1.x
- **LLM**：DeepSeek API（便宜 + 中文友好）/ OpenAI / Qwen
- **Embedding**：BGE / m3e（开源，可本地跑）
- **向量库**：Chroma（轻量）/ Milvus（生产级）
- **数据库**：MySQL 8.x
- **部署**：Docker + 阿里云学生机
